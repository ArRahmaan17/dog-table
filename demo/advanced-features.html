Advanced Features

Demonstrating URL persistence, row selection, column visibility, CSV export, and quick access to the table's live state.

Library setup

Selection, persistence, and export are enabled from the main constructor, then controlled through the table instance.

<link rel="stylesheet" href="../src/data-table.css" />

<script type="module">
  import { DataTable } from "../src/data-table.js";

  const table = new DataTable("#app", {
    data,
    rowKey: "id",
    selectable: true,
    persistence: "url",
    columns: [
      { key: "name", label: "Name" },
      { key: "region", label: "Region" },
    ],
  });

  table.init();
  table.exportCSV("dogs.csv");
</script>
