Dog Table / Custom Cells

Custom Cells and Filter Logic

This example uses DOM-node cell rendering, custom column search, and richer metadata formatting.

Library setup

Custom rendering works by returning DOM nodes and optional per-column filter logic.

<link rel="stylesheet" href="../src/data-table.css" />

<script type="module">
  import { DataTable } from "../src/data-table.js";

  const table = new DataTable("#app", {
    data: projects,
    columns: [
      { key: "dog", label: "Dog" },
      {
        key: "owner",
        label: "Owner",
        render(value) {
          return makeOwner(value);
        },
      },
      {
        key: "tags",
        label: "Tags",
        filter({ value, query }) {
          return value.join(" ").toLowerCase().includes(query);
        },
      },
    ],
  });

  table.init();
</script>
