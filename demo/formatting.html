Formatter Engine

Transforming raw data into human-readable formats using the native Intl API.

Library setup

Formatting is configured directly on each column with type, currency, locale, and format.

<link rel="stylesheet" href="../src/data-table.css" />

<script type="module">
  import { DataTable } from "../src/data-table.js";

  const table = new DataTable("#app", {
    data,
    columns: [
      { key: "name", label: "Dog Name" },
      { key: "breed", label: "Breed" },
      {
        key: "adoptedAt",
        label: "Adoption Date",
        type: "datetime",
        format: { dateStyle: "long" },
      },
      {
        key: "fee",
        label: "Fee",
        type: "money",
        currency: "USD",
      },
    ],
  });

  table.init();
</script>

Configuration Used

Notice how type: 'money' and type: 'datetime' handle the complexity of localization automatically.
