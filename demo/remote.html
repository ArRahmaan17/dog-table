Remote Data with Mocked Fetch

This demo mocks an API endpoint in the browser so you can test loading, server-side pagination, sort, and search without a backend.

Library setup

Remote mode only needs the core import plus a remote config that maps your API response.

<link rel="stylesheet" href="../src/data-table.css" />

<script type="module">
  import { DataTable } from "../src/data-table.js";

  const table = new DataTable("#app", {
    pageSize: 4,
    remote: {
      url: "/api/dogs",
      mapResponse(payload) {
        return {
          rows: payload.items,
          totalItems: payload.total,
        };
      },
    },
    columns: [
      { key: "name", label: "Name" },
      { key: "status", label: "Status" },
    ],
  });

  table.init();
</script>
