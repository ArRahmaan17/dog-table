Grouping and Row Detail

This example groups dogs by adoption region and expands details lazily only when a row is opened.

Library setup

Grouping and expandable rows are configured with groupBy and a rowDetail.render() callback.

<link rel="stylesheet" href="../src/data-table.css" />

<script type="module">
  import { DataTable } from "../src/data-table.js";

  const table = new DataTable("#app", {
    data: dogs,
    rowKey: "id",
    groupBy: "region",
    rowDetail: {
      render(row) {
        return makeDetailCard(row);
      },
    },
    columns: [
      { key: "name", label: "Name" },
      { key: "region", label: "Region" },
    ],
  });

  table.init();
</script>
